# Netflix Data Analysis using Pandas and NumPy

## Project Overview
This project analyzes Netflix user data to uncover insights and viewing patterns using Python libraries like Pandas and NumPy. The analysis includes data cleaning, exploration, and visualization to better understand user behavior and preferences.

## Features
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualizations to highlight trends and patterns
- Insights into Netflix user habits

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
